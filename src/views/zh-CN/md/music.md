# Music 音乐
----
### 基础用法
采用自定义进度条实现的音乐播放器

<div class="demo-block">
  <zmz-music :album="musicSong" :album-current="0" :volume="volume" :progress="0" :play="false" :mute="false" @on-song-next-last="onSongNext" @on-song-prev-last="onSongPrev"></zmz-music>
</div>

## Music 属性

| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| album | 音乐数组 | Array | — | [] |
| album-current | 音乐默认播放下标 | Number |  — | 0 |
| volume | 进度条位置 | Number | 0—100 | 0 |

## Music 事件
| 事件名称      | 说明          | 回调参数  |
|---------- |-------------- |-------- |
| on-pause | 暂停触发 | — |
| on-paused | 播放暂停触发 | — |
| on-play | 播放 | — |
| on-timeupdate | 监听音频时间变化 | — |
| on-ended | 检测播放是否已经结束 | function(index) |
| on-error | 监听音频错误 | — |
| on-canplay| 监听加载 | — |
| on-song-prev | 上一首 | function(index) |
| on-song-prev-last | 上一首 | function() |
| on-song-next | 下一首 | function(index) |
| on-song-next-last | 下一首 | function() |
| on-song-change | 歌曲切换 | function(index) |
